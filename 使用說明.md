# 全自動按鍵工具 — 使用說明

這個資料夾裡有兩個工具：

| 工具 | 啟動檔 | 用途 |
| --- | --- | --- |
| 單鍵連點 | `啟動連點工具.bat` | 重複連點「同一個」鍵盤鍵或滑鼠鍵 |
| 多腳本觸發器 | `啟動巨集工具.bat` | 每個觸發鍵綁一段腳本，可同時執行（進階）|

## 一、首次使用前置作業

1. 電腦需安裝 **Python 3**（[python.org](https://www.python.org/downloads/) 下載，安裝時務必勾選 **Add Python to PATH**）。
2. 雙擊對應的 `.bat`，第一次會自動安裝相依套件 `pynput`。
3. 視窗會停在畫面上；若顯示 `[ERROR] Python not found` 就是還沒裝 Python。

---

## 二、單鍵連點（啟動連點工具.bat）

1. 「捕捉按鍵」後按任一鍵盤鍵或滑鼠鍵，設為連點目標（也可用「快速選滑鼠」）。
2. 設定連點間隔（毫秒）、點擊次數上限（0 = 無限）。
3. 設定啟動/停止熱鍵（預設 F6）。
4. 按「開始連點」或按熱鍵啟動，再按一次熱鍵停止。

---

## 三、多腳本觸發器（啟動巨集工具.bat）★進階 — 美式漫畫風

可以建立**多個腳本**，每個腳本綁一個自己的**觸發鍵**。
啟用後按下某個觸發鍵 → 該腳本開始循環；再按一次 → 停止。**不同腳本可同時執行**。

### 介面：左欄管理腳本，右欄編輯腳本

- **左欄「腳本清單」**：顯示每個腳本的名稱、觸發鍵、步驟數、執行狀態。
  下方可「✚ 新增腳本 / ⧉ 複製 / ✖ 刪除」。
- **右欄「編輯器」**：編輯左欄選中的那個腳本。

### 設定一個腳本的步驟

1. 在左欄點選要編輯的腳本。
2. 設定**腳本名稱**、**觸發鍵**（按「捕捉觸發鍵」後按一個鍵盤鍵或滑鼠鍵）、**循環次數**（0 = 無限）。
3. 加入步驟（兩種方式）：
   - **錄製**：按紅色「● 錄製步驟」，接著你按的每個鍵盤鍵 / 滑鼠鍵都會記成步驟（預設按住 20ms、延遲 20ms），按 **Esc** 結束。
   - **手動**：按「✚ 新增」，選動作類型、捕捉按鍵、設定按住時間與延遲。
4. 步驟可用「編輯 / 刪除 / 上移 / 下移 / 清空」調整。

### 啟用並同時執行

1. 幫每個要用的腳本都設好**不重複**的觸發鍵與步驟。
2. 按底部綠色「**🔴 啟用觸發 (Arm)**」。
3. 之後按各腳本的觸發鍵即可開始 / 停止，多個腳本可同時運作。執行中時，左欄狀態欄會顯示**目前已跑的迴圈數**（例如「● 執行中 ×123」）。
4. 要再編輯時，先按「**■ 停用觸發 (Disarm)**」（停用會一併停止所有執行中的腳本）。

### 🛑 緊急停止鍵（Panic Key）

底部有一個**全域緊急停止鍵**，預設為 **F12**。

- **任何時候**按下它 → 立即停止所有正在執行的腳本，並自動停用觸發。腳本跑飛時不必再回視窗點停用。
- 想換鍵：按緊急停止鍵旁的「捕捉」再按你要的鍵（需先停用觸發）。
- 它**不受啟用狀態限制**、隨時生效；啟用時會檢查它不能和任何腳本的觸發鍵重複。

### ⚡ 啟用／停用快捷鍵（Arm Hotkey）

不想每次都用滑鼠點「啟用觸發」按鈕？底部有一個**全域啟用／停用快捷鍵**，預設為 **F9**。

- **按一下 = 啟用**，**再按一下 = 停用**，隨時生效（不受目前狀態限制）。
- 搭配緊急停止鍵很順手：按 **F12** 全部急停（會自動停用）→ 要再跑時按一下 **F9** 重新啟用即可。
- 想換鍵：按它旁邊的「捕捉」再按你要的鍵（需先停用觸發）。啟用時會檢查它不和觸發鍵、緊急停止鍵重複。

### 🪟 最小化小浮窗

把主視窗**最小化**後，畫面右上角會自動跳出一個**永遠置頂的小浮窗**：

- 顯示目前狀態（`○ 未啟用` / `● 已啟用 · 待命` / `● 執行中 N 個`）。
- **🛑 停止**：等同緊急停止，一鍵停掉全部。
- **▣ 開啟**：把主視窗叫回來。
- 可用滑鼠**拖曳**移動位置。把主視窗還原後小浮窗會自動收起。

> 重點提醒：
> - 觸發鍵**不能重複**，啟用時會檢查並提醒。
> - 建議觸發鍵用 **F1～F12** 這類「腳本本身不會去按」的鍵，避免腳本送出的按鍵誤觸自己。（程式已內建自我觸發防護，但用 F 鍵最保險。）
> - 右側數字鍵盤（numpad）已正確支援，顯示成「數字鍵盤1」等名稱。

### 自動記憶 / 匯入匯出

- **自動記憶**：關閉程式時會自動把所有腳本與緊急停止鍵存到同目錄的 `settings.json`，下次開啟自動載回，不必每次手動讀檔。
- **匯出 / 匯入**：「**儲存全部**」可另存成一個 `.json` 備份或分享；「**讀取**」整批叫回來。

---

## 四、小提醒

- 熱鍵是**全域**的，視窗縮小或焦點在別的程式都能觸發。
- 滑鼠動作點在**游標當下位置**；鍵盤動作作用在**目前焦點視窗**。所以開始前先把游標 / 焦點放到正確位置。
- 時間別設太短（如 0 秒），部分程式可能來不及反應。
- 部分遊戲有反作弊機制，請自行斟酌，避免違反該遊戲服務條款。
- 關閉視窗會自動清除背景監聽，不殘留系統。

## 五、打包成單一 .exe（給沒裝 Python 的電腦用）

想把「多腳本觸發器」變成免裝 Python 就能跑的單一執行檔：

1. 在這個資料夾雙擊 **`build_exe.bat`**。
2. 它會自動安裝 `pyinstaller`、`pynput` 並開始打包（`--onefile --noconsole`）。
3. 完成後，執行檔在 **`dist\全自動按鍵.exe`**，可複製到任何 Windows 電腦直接執行。
4. `settings.json` 會自動產生在 exe 同目錄，用來記住上次設定。

> 註：打包必須在 **Windows** 上進行（PyInstaller 只能產生它執行所在平台的執行檔）。

## 六、檔案結構

```

├── macro_clicker.py      多腳本觸發器 主程式
├── build_exe.bat         一鍵打包成單一 exe
├── settings.json         （自動產生）多腳本觸發器上次設定
├── 啟動連點工具.bat       啟動單鍵連點
├── 啟動巨集工具.bat       啟動多腳本觸發器
└── 使用說明.md            本說明
```
