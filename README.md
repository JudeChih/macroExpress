# 全自動按鍵工具

一套 Windows 桌面自動化小工具，包含**單鍵連點**與**多腳本觸發器**兩支程式，以 Python + tkinter + pynput 實作，無需額外框架。

---

## 功能一覽

| 工具 | 啟動方式 | 適用情境 |
|---|---|---|
| 單鍵連點 | `啟動連點工具.bat` | 快速重複按同一個鍵盤鍵或滑鼠鍵 |
| 多腳本觸發器 | `啟動巨集工具.bat` | 每個觸發鍵綁一段多步驟腳本，可同時執行 |

---

## 環境需求

- **Windows** 10 / 11
- **Python 3.8+**（[下載](https://www.python.org/downloads/)，安裝時務必勾選 **Add Python to PATH**）
- 第三方套件：`pynput`（`.bat` 首次執行時自動安裝）

> 若想在沒有 Python 的電腦上使用，可用 `build_exe.bat` 打包成單一 `.exe`，詳見下方說明。

---

## 快速開始

1. 雙擊 `啟動連點工具.bat` 或 `啟動巨集工具.bat`。
2. 首次執行會自動安裝 `pynput`，稍等數秒即可。
3. 若跳出 `[ERROR] Python not found`，請先安裝 Python 並重啟終端。

---

## 單鍵連點（autoclicker.py）

一個視窗完成所有設定：

- **連點目標**：按「捕捉按鍵」後任意按一個鍵盤鍵或滑鼠鍵，或點「快速選滑鼠」直接選左 / 中 / 右鍵。
- **連點間隔**：毫秒為單位，數字越小越快（建議 ≥ 10ms 以免目標程式來不及反應）。
- **點擊次數上限**：`0` 代表無限連點。
- **啟動 / 停止熱鍵**：預設 `F6`，可自訂；熱鍵為全域，視窗不在前景也有效。

按「開始連點」或熱鍵啟動，狀態列變綠表示執行中；再按一次熱鍵或按鈕停止。

## 多腳本觸發器（macro_clicker.py）

進階版本，支援多腳本同時執行，介面採美式漫畫風格，共 7 套可即時切換的主題。

### 基本概念

- **腳本**：一串按鍵 / 滑鼠動作的有序步驟，每個步驟可設定按住時間與延遲。
- **觸發鍵**：綁定到腳本的快捷鍵，Arm 啟用後按下觸發鍵 → 腳本開始循環；再按 → 停止。
- **Arm（啟用觸發）**：類似保險栓，啟用後熱鍵才會生效，避免編輯時誤觸。

### 建立腳本

1. 在左欄點「✚ 新增腳本」。
2. 右欄輸入**名稱**，按「捕捉觸發鍵」設定觸發鍵，填入**循環次數**（`0` = 無限）。
3. 加入步驟，兩種方式：
   - **錄製**：按紅色「● 錄製步驟」，操作你要的按鍵序列，按 `Esc` 結束。每步預設按住 20ms、延遲 20ms。
   - **手動**：按「✚ 新增」，選動作類型（鍵盤 / 各滑鼠鍵）並設定時間參數。
4. 步驟可任意**編輯 / 刪除 / 上移 / 下移 / 清空**。

### 執行腳本

- 按底部「**🔴 啟用觸發 (Arm)**」後，按各腳本的觸發鍵即可單獨開關。
- 按「**🚀 一鍵全部啟動**」或快捷鍵（預設 `F8`），同時跑所有有步驟的腳本。
- 左欄狀態欄即時顯示迴圈計數（例：`● 執行中 ×42`）。

### 全域快捷鍵

| 快捷鍵 | 預設 | 功能 |
|---|---|---|
| 緊急停止 | `F12` | 立即停止所有腳本並自動 Disarm |
| 啟用 / 停用 | `F9` | 切換 Arm 狀態 |
| 一鍵全部啟動 | `F8` | 同時啟動所有腳本 |

三個快捷鍵均為全域，任何視窗下皆有效，可在工具底部自訂。啟用時會自動檢查是否有衝突。

### 最小化浮窗

主視窗最小化後，畫面右上角自動出現永遠置頂的小浮窗，顯示執行狀態並提供緊急停止與還原按鈕，支援滑鼠拖曳。

### 自動記憶

關閉程式時自動將所有腳本、快捷鍵設定與主題選擇儲存至 `settings.json`，下次開啟自動還原。也可透過「**儲存全部**」手動匯出備份，「**讀取**」匯入。

## 打包成單一 .exe

在沒有 Python 的 Windows 電腦上直接執行：

1. 雙擊 **`build_exe.bat`**，自動安裝 `pyinstaller` 並打包（`--onefile --noconsole`）。
2. 完成後執行檔位於 `dist\全自動按鍵.exe`，可獨立複製到其他電腦。
3. `settings.json` 會自動產生在 exe 同目錄。

> 注意：打包只能在 **Windows** 上進行。

---

## 注意事項

- 熱鍵為**全域監聽**，在任何視窗下都有效；關閉程式後監聽器會確實清除，不殘留系統資源。
- 滑鼠動作作用於**游標當下位置**；鍵盤動作作用於**當前焦點視窗**。請在啟動前先確認游標 / 焦點位置正確。
- 建議觸發鍵使用 `F1`～`F12` 等功能鍵，避免腳本送出的按鍵誤觸觸發鍵（程式已內建自我觸發防護，但功能鍵最保險）。
- 右側數字鍵盤（Numpad）已正確支援，顯示為「數字鍵盤0」～「數字鍵盤9」等名稱。
- 部分遊戲有反作弊機制，請自行評估風險，避免違反服務條款。

```
全自動按鍵/
├── autoclicker.py        單鍵連點主程式
├── macro_clicker.py      多腳本觸發器主程式
├── build_exe.bat         一鍵打包成單一 exe
├── settings.json         （自動產生）多腳本觸發器設定
├── 啟動連點工具.bat       啟動單鍵連點
├── 啟動巨集工具.bat       啟動多腳本觸發器
└── 使用說明.md            詳細操作說明
```
